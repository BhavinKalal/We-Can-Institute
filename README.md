# WE CAN Institute

WE CAN Institute is a modern institute website with a public-facing website, an admin panel, and a backend content system.

It is built to manage institute content dynamically instead of relying on static hardcoded sections.

## Website Overview

The project includes:

- `frontend/` - public website
- `admin-frontend/` - admin panel
- `backend/` - backend API and data layer

## Main Sections

The website supports dynamic content for:

- Hero section
- Hero statistics
- Site settings
- Batches and programs
- Faculty
- Testimonials
- Gallery
- Blog
- Enquiries

## Admin Capabilities

The admin panel is used to manage:

- Homepage content
- Contact and social details
- Batch information
- Faculty profiles
- Testimonials
- Gallery items
- Blog posts
- Enquiry records
- Admin users

## Media Support

The website supports uploaded media for:

- Hero poster and video
- Faculty photos
- Gallery images and videos
- Blog cover images

## Purpose

This project is designed for an institute that wants:

- a professional website
- a manageable admin panel
- dynamic content updates
- enquiry collection
- a scalable structure for future improvements
